# Reveal.js Formal Template

## How to use

1. Click "Use this template" and create new repository based on this.
1. Clone it and run ``npm install``.
1. Fix ``src/config.pug``.
1. Run ``npm run develop``.
1. Edit ``src/slides.pug``.
